## O que é

**Você Lavaria?** é um jogo educativo interativo sobre prevenção à lavagem de dinheiro (PLD) e financiamento do terrorismo (FT), com foco no ecossistema brasileiro de fintechs, Pix e criptoativos.

O jogador escolhe entre dois lados:

- **🔍 Investigador** — você é analista de PLD em uma fintech. Recebe alertas reais, analisa dados de clientes e transações, e decide: escalar, solicitar documentos ou comunicar ao COAF. Cada decisão tem consequências.
- **🏗️ Construtor** — você monta o esquema, escolhendo canais, laranjas, empresas e criptoativos. Um medidor de exposição sobe a cada decisão. Você aprende por que nenhum esquema é invisível.

## Conteúdo

- **100 missões únicas** — 50 por modo, em três níveis de dificuldade (Fácil, Médio, Difícil)
- **7 denominações de notas de real** caindo animadas na tela inicial
- **Briefing interativo** antes de cada missão: identifique red flags, ordene fluxos, escolha hipóteses
- **Debriefing completo**: mapa animado do esquema, red flags encontrados/perdidos, base legal com explicação em linguagem simples
- **Glossário permanente** com 20 termos de PLD/FT sempre visível durante o jogo
- **Sistema de streak** com som e badge visual a cada acerto consecutivo
- **10 conquistas** desbloqueáveis por progresso
- **Tela de estatísticas pessoais** com taxa de acerto, pontuação total e melhor sequência
- **Sons opcionais** via Web Audio API (sem arquivo externo)
- **Badges de operação real** em 18 missões — com link direto para fonte oficial no gov.br
- **Progresso salvo** automaticamente via localStorage

## Base legal e regulatória

Todas as missões têm referência a normas reais:

- **Lei nº 9.613/1998** — Lei de Lavagem de Dinheiro
- **Lei nº 12.683/2012** — Atualização da Lei de Lavagem
- **Circular BCB nº 3.978/2020** — Política de PLD/FT para instituições autorizadas
- **Carta Circular BCB nº 4.001/2020** — Catálogo de red flags regulatórios
- **Resolução BCB nº 1/2020** — Regulamento do Pix e MED
- **Resolução BCB nº 80/2021** — Instituições de Pagamento
- **Lei nº 14.478/2022** — Marco legal de ativos virtuais
- **Resolução BCB nº 520/2025** — PSAVs (prestadoras de serviços de ativos virtuais)
- **FATF/GAFI** — Relatórios de tipologias e red flags (2010–2026)
- **FinCEN** — Advisories sobre CVC, ransomware, money mules e pig butchering
- **Europol** — Inteligência sobre money mules e criptoativos

## Tecnologia

Projeto 100% autocontido — sem dependências externas além das fontes do Google Fonts.

- HTML5 + CSS3 + JavaScript vanilla
- Canvas API para a animação das notas
- Web Audio API para sons (sem arquivo externo)
- LocalStorage para persistência de progresso
- Nenhum framework, nenhum backend, nenhuma dependência de pacote
